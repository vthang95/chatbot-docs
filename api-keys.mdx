To start building applications.\
You first have to create an API Key.


### Create API Key
1. Go to [https://platform.minostack.com/api-keys](https://platform.minostack.com/api-keys)
2. Click on `Create API Key`
3. Enter a name for your API Key. You can set expiration date for the key.
4. Click on `Create API Key`. If successful, you will see your API Key. Please save it somewhere safe. Because this is the only time you will be able to see it.
5. Use the API Key in your application.


### Revoke API Key
1. Go to [https://platform.minostack.com/api-keys](https://platform.minostack.com/api-keys)
2. Click on `Revoke` button next to the API Key you want to revoke.
3. Confirm the revocation.
